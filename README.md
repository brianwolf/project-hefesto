# :card_index_dividers: Hefesto

Dios herrero griego usado para forjar tus proyectos

![img](img/hefesto.jpg)

## :gear: Requisitos

* python 3.9
* virtualenv

## :tada: Uso

Levantar el ambiente

```bash
virtualenv -p python3.9 env

. env/bin/activate

pip install -r requeriments.txt
```

Ejecutar

```bash
python app.py
```
